# LZ77 compressor/decompressor

Учебная реализация алгоритма сжатия LZ77 на чистом C (стандарт C90).

## Возможности
- Сжатие и распаковка текстовых / бинарных файлов
- Скользящее окно 4096 байт, максимальная длина совпадения 18
- Полностью обратимо: исходный файл после цикла сжатие‑распаковка совпадает побайтово



## Сборка                        основные 
компилутор     стандарт С90   предупреждения   ещё предупреждения    
gcc              -std=c90       -pedantic             -Wall         lz77.c   -o    lz77

gcc -std=c90 -pedantic -Wall lz77.c -o lz77


## сжатие
 
./lz77 -c input.txt compressed.lz

вывод:  Сжатие: 12345 -> 9876 байтов (80.00%)



## Распаковка

./lz77 -d compressed.lz restored.txt

ВЫВОД: разжатие в байтах: 12345 



## Проверка на совпадение

diff input.txt restored.txt    # на Linux
fc /b input.txt restored.txt   # на Windows
